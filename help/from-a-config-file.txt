read config from a file
args:
  -f,--file options,set the config file
  -h,--help options,get the cmd help
examples: 
  run as shell args
bash ./from-a-config-file.sh
  run as runable application
./from-a-config-file.sh --file=a-config-file-2.txt 

  without args: 
./from-a-config-file.sh 
  with args: 
./from-a-config-file.sh --file=a-config-file-2.txt

  with debug mode: 
./from-a-config-file.sh --debug
